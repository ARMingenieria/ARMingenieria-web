# Subida a GitHub Pages

1. En el repositorio, elimina los archivos antiguos si quieres dejarlo limpio.
2. Descomprime este ZIP.
3. Sube TODOS los archivos sueltos a la raíz del repositorio. No subas la carpeta completa ni el ZIP.
4. Deben verse directamente en GitHub: `index.html`, `styles.css`, `app.js`, `arm-ingenieria-logo.png`, `og-arm-ingenieria.jpg`, `robots.txt`, `sitemap.xml`, etc.
5. Ve a Settings → Pages y confirma: Branch `main`, carpeta `/ (root)`.
6. Espera 2-5 minutos y abre: https://armingenieria.github.io/ARMingenieria-web/
